## git命令使用

- git init 初始化整个仓库

### 提交本地仓库到远程库

- git remote add origin (gitssh)   // 连接本地库到远程库中
- git add .
- git commit -m "备注的信息"
- git push -u origin master （第一次仓库为空时要加-u参数）
- git push origin master (后续添加可以直接push)

### 提交分支到远程库中

- git branch (对应的分支名字 dev)
- git checkout dev  // 切换到当前的dev分支
- // git branch  查看当前的分支名字
- git switch dev
