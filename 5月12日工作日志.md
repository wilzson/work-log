## git add 添加文件

git add <file>  提交单个文件

git add .   // 提交所有发生变化的文件添加到暂存区(增改)，不包括删除

git add -u  // 只提交修改的文件和删除添加到暂存区，不包括新文件

git add -A  //所有文件（. && -u 集合）  git的版本不同，命令则不一样
